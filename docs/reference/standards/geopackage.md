# GeoPackage (`gpkg`) Standard

**GeoPackage** is an official [OGC Encoding Standard](https://www.ogc.org/standards/geopackage/) that describes a set of rules and conventions for storing vector features,imagery tile matrix sets, raster map tile matrix sets and non-spatial tabular data in a SQLite database. The standard also describes rules for extending the capabilities of a GeoPackage.

The standard is particularly relevant for this project in that the entire project revolves around the original nationwide US parcel data which is delivered in a single compressed `gpkg` file.
