# Configuration

- [config.yml](./config.yml): Project configuration specific to R and the R [config](https://github.com/r-lib/config) package.
- [rclone.conf](./rclone.conf): Rclone configuration for remote storage (redacted keys)
- [gdalrc](./gdalrc): GDAL configuration for remote storage and memory/session settings.
- [postgresql.conf](./postgresql.conf): PostgreSQL configuration
- [tileserver.toml](./tileserver.toml): [tileserver-rs](https://github.com//tileserver-rs) configuration
- [traefik.toml](./traefik.toml): [Traefik](https://traefik.io/) configuration
